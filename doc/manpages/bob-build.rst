bob-build
=========

.. only:: not man

   Name
   ----

   bob-build - Bob release mode build

Synopsis
--------

::

    bob build [-h] [--destination DEST] [-f] [-n] [-b | -B]
              [--clean | --incremental] [--resume] [-q] [-v] [-D DEFINES]
              [-c CONFIGFILE] [-e NAME] [-E] [--upload] [--download MODE]
              [--sandbox | --no-sandbox] [--clean-checkout]
              PACKAGE [PACKAGE ...]

Description
-----------

The *bob build* command is building packages locally in release mode.
